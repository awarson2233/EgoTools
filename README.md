## 📖 使用方法
1. 前往 Release 发布页面下载最新本的压缩包；
2. 解压后运行文件夹中的 EgoTools.exe 即可。

## 📌 已完成的功能
  
| 状态 |          功能                        |
|:--------:|:-------------------------------:|
|    ✅     | 设备信息查看                    |
|    ✅     | 键盘分离设置                    |
|    ✅     | 屏幕色彩管理                    |
|    ✅     | 电池阈值设置                    |
